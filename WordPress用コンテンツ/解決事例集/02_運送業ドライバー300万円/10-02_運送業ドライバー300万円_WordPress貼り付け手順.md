# WordPressへの貼り付け手順書

## 【事例2】運送業ドライバー300万円回収ページの作成方法

この手順書では、HTMLとCSSをコピペするだけでWordPressページを作成する方法を説明します。

---

## 📋 準備するファイル

- `10-02_運送業ドライバー300万円_HTML.html`（HTMLコード）
- `10-02_運送業ドライバー300万円_CSS.css`（CSSコード）

---

## ステップ1: CSSの追加（全ページで1回のみ）

まず、サイト全体に適用されるCSSを追加します。**これは最初の1回だけ実行すればOKです。**

### 手順：

1. WordPress管理画面にログイン
2. 左メニューから **「外観」→「カスタマイズ」** をクリック
3. カスタマイザー画面の左メニューから **「追加CSS」** をクリック
4. `10-02_運送業ドライバー300万円_CSS.css` ファイルを開く
5. **CSS全体をコピー**（Ctrl+A → Ctrl+C または Cmd+A → Cmd+C）
6. WordPressの「追加CSS」エリアに **ペースト**（Ctrl+V または Cmd+V）
7. 右上の **「公開」** ボタンをクリック

これでCSSの設定は完了です。

---

## ステップ2: 新規ページの作成

次に、事例詳細ページを作成します。

### 手順：

1. WordPress管理画面で、左メニューから **「固定ページ」→「新規追加」** をクリック
2. ページタイトル欄に以下を入力：
   ```
   【事例2】運送業ドライバーが就業規則にない固定残業代を無効にし約300万円を回収
   ```

---

## ステップ3: HTMLコードの貼り付け

カスタムHTMLブロックを使って、作成済みのHTMLコードを貼り付けます。

### 手順：

1. ブロックエディタ画面で、左上の **「＋」ボタン** をクリック
2. 検索ボックスに **「カスタムHTML」** と入力して検索
3. **「カスタムHTML」ブロック** を選択（黒い背景のアイコン）
4. `10-02_運送業ドライバー300万円_HTML.html` ファイルを開く
5. **HTML全体をコピー**（Ctrl+A → Ctrl+C または Cmd+A → Cmd+C）
6. WordPress編集画面の **カスタムHTMLブロック** 内に **ペースト**（Ctrl+V または Cmd+V）

---

## ステップ4: プレビューで確認

HTMLを貼り付けたら、見た目を確認します。

### 手順：

1. 右上の **「プレビュー」** ボタンをクリック
2. **「新しいタブでプレビュー」** を選択
3. デザインが正しく表示されているか確認

### 確認ポイント：

✓ セクションごとに色分けされている
✓ 見出しにアンダーラインが表示されている
✓ ボタン（電話・メール・LINE）が正しく配置されている
✓ スマホサイズでも正しく表示される（ブラウザの開発者ツールで確認可能）

---

## ステップ5: ページ設定

公開前に、ページの基本設定を行います。

### 手順：

1. 右側のサイドバーで **「ページ」タブ** を選択
2. **「URL」** を設定（例：`case-study-driver-300`）
   - わかりやすく、SEOに配慮したURLにします
3. **「親ページ」** を設定（もし「解決事例一覧」ページがあれば、それを親に設定）
4. **「抜粋」** を入力（検索結果やリンクに表示される概要文）：
   ```
   運送業ドライバーが就業規則に記載のない固定残業代を無効にし、約300万円の未払い残業代を回収した解決事例。証拠資料の確保から交渉戦略まで詳しく解説します。
   ```

---

## ステップ6: SEO設定（推奨）

SWELLの場合、SEO設定機能があるので活用しましょう。

### 手順：

1. 下にスクロールして **「SEO設定」** セクションを探す
2. **「メタディスクリプション」** を入力：
   ```
   運送業ドライバーが就業規則に記載のない固定残業代を無効にし、約300万円を回収した事例。福岡県弁護士会所属・川波晃生弁護士が詳しく解説。
   ```
3. **「メタキーワード」** を入力（任意）：
   ```
   残業代請求, 運送業, ドライバー, 固定残業代, 解決事例, 福岡, 弁護士
   ```

---

## ステップ7: リンクの設定

ページ内のリンク（「#」で始まるリンク）を実際のURLに変更します。

### 変更が必要な箇所：

HTMLコード内の以下のリンクを、実際のページURLに置き換えてください：

```html
<!-- 変更前 -->
<a href="#">今すぐ無料相談する（電話）</a>
<a href="#">メールで相談する</a>
<a href="#">LINE で相談する</a>
<a href="#">解決事例一覧に戻る</a>
<a href="#">残業代請求の基礎知識</a>
<a href="#">固定残業代とは？</a>
<a href="#">弁護士費用について</a>

<!-- 変更後の例 -->
<a href="tel:05054823433">今すぐ無料相談する（電話）</a>
<a href="mailto:info@lawyer-ak.com">メールで相談する</a>
<a href="https://line.me/ti/p/あなたのLINE_ID">LINE で相談する</a>
<a href="/case-studies">解決事例一覧に戻る</a>
<a href="/knowledge">残業代請求の基礎知識</a>
<a href="/fixed-overtime">固定残業代とは？</a>
<a href="/fee">弁護士費用について</a>
```

### 変更方法：

1. カスタムHTMLブロック内で、該当箇所を検索（Ctrl+F または Cmd+F）
2. `href="#"` を検索して、実際のURLに置き換える
3. 電話番号リンクは `tel:05054823433`（ハイフンなし）
4. メールリンクは `mailto:info@lawyer-ak.com`
5. その他のリンクは、実際のページURLに変更

---

## ステップ8: 公開

すべての設定が完了したら、ページを公開します。

### 手順：

1. 右上の **「公開」** ボタンをクリック
2. 確認画面が表示されたら、もう一度 **「公開」** をクリック
3. **「ページを表示」** をクリックして、実際のページを確認

---

## トラブルシューティング

### デザインが反映されない場合

1. **CSSが正しく追加されているか確認**
   - 「外観」→「カスタマイズ」→「追加CSS」
   - CSSコードが貼り付けられているか確認
   - 「公開」ボタンを押したか確認

2. **キャッシュをクリア**
   - ブラウザのキャッシュをクリア（Ctrl+Shift+Del または Cmd+Shift+Del）
   - WordPressのキャッシュプラグインを使っている場合は、そちらもクリア

3. **HTMLブロックを確認**
   - カスタムHTMLブロックが正しく選択されているか
   - HTMLコードが全てコピーされているか

### レイアウトが崩れる場合

1. **SWELLテーマの設定を確認**
   - 「外観」→「カスタマイズ」→「コンテンツの設定」
   - 「記事・固定ページ」の設定で、不要な装飾がオフになっているか確認

2. **ブロックの順序を確認**
   - カスタムHTMLブロックが1つだけで、他のブロックと混在していないか確認

### ボタンがクリックできない場合

1. **リンクのURLが正しく設定されているか確認**
   - ステップ7で説明したリンク設定を確認

2. **電話番号リンクの形式を確認**
   - `tel:05054823433`（ハイフンなしで統一）

---

## 次のステップ

ページが公開できたら、以下を実施しましょう：

1. **「解決事例一覧」ページからリンクを追加**
   - 一覧ページに、このページへのリンクを追加

2. **トップページのリンクを確認**
   - トップページの「解決事例」セクションから、このページへアクセスできるか確認

3. **内部リンクの設置**
   - 関連するページ（料金ページ、よくある質問など）からもリンクを追加

4. **スマホでの表示確認**
   - 実際のスマートフォンで表示を確認
   - ボタンのタップが正しく動作するか確認

---

## まとめ

この手順で、プロフェッショナルなデザインの事例詳細ページを簡単に作成できます。

**今後、他の事例ページを作成する場合も、同じ手順でOKです。**

- CSSの追加は最初の1回だけ
- HTMLファイルの内容を変更して、コピペするだけ
