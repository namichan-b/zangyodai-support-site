# モックアップファイル一覧

このフォルダには、残業代サポートサイトのモックアップHTML/CSSファイルが格納されています。

## ファイル構成

### メインモックアップファイル（完成版）
- `mockup-zangyodai-support.html` - **トップページ**
- `mockup-flow.html` - **相談の流れ**
- `mockup-lawyer.html` - **弁護士紹介**
- `mockup-fee.html` - **費用・料金**
- `mockup-cases.html` - **解決事例**

### 開発用・テスト版ファイル

#### 1. 相談の流れページ（各種バージョン）
- `02_相談の流れ_LiveServer_プレビュー.html` - 完成版プレビュー（スマホ対応済み）
- `02_相談の流れ_HTML_ミドルブルー.html` - ミドルブルー配色版
- `02_相談の流れ_HTML.html` - 基本版
- `02_相談の流れ_HTML_ミドルブルー.html.bak2` - バックアップ

#### 2. ヒーローエリア（トップページのメインビジュアル）
- `ヒーローエリア_シンプル版.html` - シンプルなデザイン（中央揃え）
- `ヒーローエリア_改善版.html` - 左揃えテキスト版（参考用）

#### 3. 弁護士紹介ページ（テスト版）
- `弁護士紹介_左揃え版.html` - 左揃えレイアウト版

## 使用方法

各HTMLファイルは、ブラウザで直接開いて確認できます。
Live Serverなどの拡張機能を使用すると、リアルタイムプレビューも可能です。

## 注意事項

- これらはモックアップファイルのため、実際のWordPressへの実装時には適切な変換が必要です
- CSSはHTML内に埋め込まれています
- レスポンシブデザイン対応済み（スマートフォン、タブレット、PC）
