# WordPressへの貼り付け手順書

## 「費用・料金」ページの作成方法 - リッチデザイン版

この手順書では、HTMLとCSSをコピペするだけでWordPressページを作成する方法を説明します。

---

## 📋 準備するファイル

- `03_費用・料金_HTML.html`（HTMLコード）
- `03_費用・料金_CSS.css`（CSSコード）

---

## ステップ1: CSSの追加（全ページで1回のみ）

まず、サイト全体に適用されるCSSを追加します。

### 手順：

1. WordPress管理画面にログイン
2. 左メニューから **「外観」→「カスタマイズ」** をクリック
3. カスタマイザー画面の左メニューから **「追加CSS」** をクリック
4. 既存のCSS（トップページ用・相談の流れページ用）の **一番下** までスクロール
5. 最後に、以下のコメントを追加：

```css
/* ============================================
   費用・料金ページ用CSS
   ============================================ */
```

6. `03_費用・料金_CSS.css` ファイルを開く
7. **CSS全体をコピー**（Ctrl+A → Ctrl+C または Cmd+A → Cmd+C）
8. WordPressの「追加CSS」エリアに **ペースト**（Ctrl+V または Cmd+V）
9. 右上の **「公開」** ボタンをクリック

これでCSSの設定は完了です。

## ステップ2: 新規ページの作成

次に、「費用・料金」ページを作成します。

### 手順：

1. WordPress管理画面で、左メニューから **「固定ページ」→「新規追加」** をクリック
2. ページタイトル欄に以下を入力：
   ```
   費用・料金
   ```

---

## ステップ3: HTMLコードの貼り付け

カスタムHTMLブロックを使って、作成済みのHTMLコードを貼り付けます。

### 手順：

1. ブロックエディタ画面で、左上の **「＋」ボタン** をクリック
2. 検索ボックスに **「カスタムHTML」** と入力して検索
3. **「カスタムHTML」ブロック** を選択（黒い背景のアイコン）
4. `03_費用・料金_HTML.html` ファイルを開く
5. **HTML全体をコピー**（Ctrl+A → Ctrl+C または Cmd+A → Cmd+C）
6. WordPress編集画面の **カスタムHTMLブロック** 内に **ペースト**（Ctrl+V または Cmd+V）

---

## ステップ4: プレビューで確認

HTMLを貼り付けたら、見た目を確認します。

### 手順：

1. 右上の **「プレビュー」** ボタンをクリック
2. **「新しいタブでプレビュー」** を選択
3. デザインが正しく表示されているか確認

### 確認ポイント：

✓ ヘッダーがグラデーションブルーで表示されている
✓ 3つのメインポイントカード（初回相談0円、着手金0円、完全成功報酬27.5%）が表示されている
✓ 料金テーブルが2カラムで表示されている（交渉段階、訴訟段階）
✓ 料金シミュレーションが4カラムで表示されている
✓ 比較テーブルが正しく表示されている
✓ CTAボタン（お問い合わせフォーム・電話番号）が表示されている
✓ スマホサイズでも正しく表示される

## ステップ5: ページ設定

公開前に、ページの基本設定を行います。

### 手順：

1. 右側のサイドバーで **「ページ」タブ** を選択
2. **「URL」** を設定（例：`fee` または `price`）
   - わかりやすく、SEOに配慮したURLにします
3. **「親ページ」** を設定（必要に応じて）
4. **「抜粋」** を入力（検索結果やリンクに表示される概要文）：
   ```
   完全成功報酬制で安心。初回相談無料・着手金0円でご利用いただけます。明確な料金体系と料金シミュレーションをご確認ください。
   ```

## ステップ7: リンクの確認

ページ内のリンクを確認します。

### 確認が必要な箇所：

HTMLコード内の以下のリンクが正しいか確認してください：

```html
<!-- お問い合わせフォームURL -->
<a href="https://zangyodai-support.com/%e3%81%8a%e5%95%8f%e3%81%84%e5%90%88%e3%82%8f%e3%81%9b/">
    お問い合わせフォームへ
</a>

<!-- 電話番号リンク -->
<a href="tel:05054823433">050-5482-3433</a>
```

### 変更方法：

1. カスタムHTMLブロック内で、該当箇所を検索（Ctrl+F または Cmd+F）
2. URLが正しいか確認
3. 電話番号リンク `tel:05054823433` はそのままでOK
4. お問い合わせフォームURLが正しいか確認

## トラブルシューティング

### デザインが反映されない場合

1. **CSSが正しく追加されているか確認**
   - 「外観」→「カスタマイズ」→「追加CSS」
   - 費用・料金ページ用のCSSコードが貼り付けられているか確認
   - 「公開」ボタンを押したか確認

2. **キャッシュをクリア**
   - ブラウザのキャッシュをクリア（Ctrl+Shift+Del または Cmd+Shift+Del）
   - WordPressのキャッシュプラグインを使っている場合は、そちらもクリア

3. **HTMLブロックを確認**
   - カスタムHTMLブロックが正しく選択されているか
   - HTMLコードが全てコピーされているか

### レイアウトが崩れる場合

1. **SWELLテーマの設定を確認**
   - 「外観」→「カスタマイズ」→「コンテンツの設定」
   - 「記事・固定ページ」の設定で、不要な装飾がオフになっているか確認

2. **ブロックの順序を確認**
   - カスタムHTMLブロックが1つだけで、他のブロックと混在していないか確認

### 比較テーブルが正しく表示されない場合

1. **HTMLのテーブルタグが正しく記述されているか確認**
   - `<table>` と `</table>` が正しくペアになっているか
   - `<thead>` と `<tbody>` が正しく配置されているか

2. **スマホでの表示を確認**
   - スマホでは横スクロールで全体が見えるようになっています
   - テーブルが画面幅に収まらない場合は正常です

### ボタンがクリックできない場合

1. **リンクのURLが正しく設定されているか確認**
   - ステップ7で説明したリンク設定を確認

2. **電話番号リンクの形式を確認**
   - `tel:05054823433`（ハイフンなしで統一）

## よくある質問

### Q. CSSは毎回追加する必要がありますか？

A. いいえ、CSSは最初の1回だけ追加すればOKです。次回以降、他のページを作る場合も、この「費用・料金」用のCSSは残しておいてください。

### Q. 料金を変更したい場合は？

A. HTMLコード内の料金部分（27.5%や33%、計算例の数字など）を直接編集してください。変更後、ページを更新すれば反映されます。

### Q. 電話番号を変更したい場合は？

A. HTMLコード内の `tel:05054823433` と `050-5482-3433` を一括置換で変更してください。ハイフンあり/なし両方を変更する必要があります。

### Q. 比較テーブルの内容を変更できますか？

A. はい、HTMLコード内の`<td>`タグ内のテキストを編集することで変更できます。ただし、テーブル構造は維持してください。

### Q. シミュレーション例を追加/削除したい場合は？

A. HTMLコード内の`<div class="fee-calculator-example">`ブロックをコピーして追加、または削除してください。レスポンシブデザインは自動調整されます。
