# WordPressへの貼り付け手順書

## 「相談の流れ」ページの作成方法 - リッチデザイン版

この手順書では、HTMLとCSSをコピペするだけでWordPressページを作成する方法を説明します。

---

## 📋 準備するファイル

- `02_相談の流れ_HTML.html`（HTMLコード）
- `02_相談の流れ_CSS.css`（CSSコード）

---

## ステップ1: CSSの追加（全ページで1回のみ）

まず、サイト全体に適用されるCSSを追加します。

### 手順：

1. WordPress管理画面にログイン
2. 左メニューから **「外観」→「カスタマイズ」** をクリック
3. カスタマイザー画面の左メニューから **「追加CSS」** をクリック
4. 既存のCSS（トップページ用・事例詳細用）の **一番下** までスクロール
5. 最後に、以下のコメントを追加：

```css
/* ============================================
   相談の流れページ用CSS
   ============================================ */
```

6. `02_相談の流れ_CSS.css` ファイルを開く
7. **CSS全体をコピー**（Ctrl+A → Ctrl+C または Cmd+A → Cmd+C）
8. WordPressの「追加CSS」エリアに **ペースト**（Ctrl+V または Cmd+V）
9. 右上の **「公開」** ボタンをクリック

これでCSSの設定は完了です。

## ステップ2: 新規ページの作成

次に、「相談の流れ」ページを作成します。

### 手順：

1. WordPress管理画面で、左メニューから **「固定ページ」→「新規追加」** をクリック
2. ページタイトル欄に以下を入力：
   ```
   相談の流れ
   ```

---

## ステップ3: HTMLコードの貼り付け

カスタムHTMLブロックを使って、作成済みのHTMLコードを貼り付けます。

### 手順：

1. ブロックエディタ画面で、左上の **「＋」ボタン** をクリック
2. 検索ボックスに **「カスタムHTML」** と入力して検索
3. **「カスタムHTML」ブロック** を選択（黒い背景のアイコン）
4. `02_相談の流れ_HTML.html` ファイルを開く
5. **HTML全体をコピー**（Ctrl+A → Ctrl+C または Cmd+A → Cmd+C）
6. WordPress編集画面の **カスタムHTMLブロック** 内に **ペースト**（Ctrl+V または Cmd+V）

---

## ステップ4: プレビューで確認

HTMLを貼り付けたら、見た目を確認します。

### 手順：

1. 右上の **「プレビュー」** ボタンをクリック
2. **「新しいタブでプレビュー」** を選択
3. デザインが正しく表示されているか確認

### 確認ポイント：

✓ ヘッダーがグラデーションブルーで表示されている
✓ 6つのステップが明確に区切られている
✓ CTAボタン（電話・LINE・フォーム）が3列に配置されている
✓ タイムライン（進行イメージ）が横並びで表示されている
✓ 資料チェックリストが開閉できる（details要素）
✓ FAQが開閉できる
✓ スマホサイズでも正しく表示される

## ステップ5: ページ設定

公開前に、ページの基本設定を行います。

### 手順：

1. 右側のサイドバーで **「ページ」タブ** を選択
2. **「URL」** を設定（例：`consultation-flow` または `soudan-flow`）
   - わかりやすく、SEOに配慮したURLにします
3. **「親ページ」** を設定（必要に応じて）
4. **「抜粋」** を入力（検索結果やリンクに表示される概要文）：
   ```
   未払い残業代の回収まで、6つのステップで安心サポート。ご相談から解決までの流れをわかりやすくご説明します。初回相談無料、全国対応。
   ```

## ステップ7: リンクの設定

ページ内のリンク（「#」で始まるリンク）を実際のURLに変更します。

### 変更が必要な箇所：

HTMLコード内の以下のリンクを、実際のページURLに置き換えてください：

```html
<!-- 変更前 -->
<a href="#">LINEで相談</a>
<a href="#">友だち追加はこちら</a>
<a href="#">相談予約フォーム</a>
<a href="#">相談予約する</a>

<!-- 変更後の例 -->
<a href="https://line.me/ti/p/あなたのLINE_ID">LINEで相談</a>
<a href="https://line.me/ti/p/あなたのLINE_ID">友だち追加はこちら</a>
<a href="/contact">相談予約フォーム</a>
<a href="/contact">相談予約する</a>
```

### 変更方法：

1. カスタムHTMLブロック内で、該当箇所を検索（Ctrl+F または Cmd+F）
2. `href="#"` を検索して、実際のURLに置き換える
3. 電話番号リンク `tel:05054823433` はそのままでOK
4. メールリンク `mailto:info@lawyer-ak.com` もそのままでOK
5. LINEリンクは、あなたのLINE公式アカウントのURLに変更
6. フォームリンクは、お問い合わせページのURLに変更

### 一括置換の方法（推奨）：

1. カスタムHTMLブロック内で Ctrl+F（または Cmd+F）
2. `href="#"` を検索
3. 該当箇所を1つずつ確認して、適切なURLに変更

すべての設定が完了したら、ページを公開します。

## デザインの特徴

この「相談の流れ」ページは、以下のリッチなデザイン要素を含んでいます：

### 1. **ヘッダーセクション**
- グラデーションブルーの背景
- 大きなタイトルと説明文
- 目を引くデザイン

### 2. **CTAボタン（3種類）**
- 電話（緑のグラデーション）
- LINE（LINEグリーン）
- フォーム（ブルーグラデーション）
- ホバーエフェクト付き

### 3. **タイムライン（進行イメージ）**
- 6つのステップが横並び
- 矢印で流れを表示
- スマホでは縦並びに自動変換

### 4. **各ステップセクション**
- ブルーのヘッダー
- ゴールドのステップ番号
- ホワイトの本文エリア
- ホバーで浮き上がる効果

### 5. **連絡方法カード**
- 電話・LINE・フォームの3カード
- アイコン＋ボタン付き
- ホバーでアニメーション

### 6. **資料チェックリスト**
- 開閉可能な詳細情報（details要素）
- 職種別に分類
- チェックボックス形式

### 7. **アドバイスボックス**
- 黄色の背景（アドバイス）
- 青色の背景（情報）
- 電球アイコン付き

### 8. **方針カード**
- 交渉・労働審判・訴訟の3カード
- 期間と対象者を明記
- ホバーで拡大効果

### 9. **FAQセクション**
- 開閉可能なアコーディオン
- 質問をクリックで回答表示
- +/- アイコンで視覚的にわかりやすい

### 10. **完全レスポンシブ**
- PC・タブレット・スマホに最適化
- 3カラム→2カラム→1カラムに自動変換
- タイムラインも縦並びに変更

## トラブルシューティング

### デザインが反映されない場合

1. **CSSが正しく追加されているか確認**
   - 「外観」→「カスタマイズ」→「追加CSS」
   - 相談の流れページ用のCSSコードが貼り付けられているか確認
   - 「公開」ボタンを押したか確認

2. **キャッシュをクリア**
   - ブラウザのキャッシュをクリア（Ctrl+Shift+Del または Cmd+Shift+Del）
   - WordPressのキャッシュプラグインを使っている場合は、そちらもクリア

3. **HTMLブロックを確認**
   - カスタムHTMLブロックが正しく選択されているか
   - HTMLコードが全てコピーされているか

### レイアウトが崩れる場合

1. **SWELLテーマの設定を確認**
   - 「外観」→「カスタマイズ」→「コンテンツの設定」
   - 「記事・固定ページ」の設定で、不要な装飾がオフになっているか確認

2. **ブロックの順序を確認**
   - カスタムHTMLブロックが1つだけで、他のブロックと混在していないか確認

### FAQや資料チェックリストが開閉しない場合

1. **HTMLの`<details>`タグが正しく記述されているか確認**
   - `<details>` と `</details>` が正しくペアになっているか
   - `<summary>` タグが正しく配置されているか

2. **ブラウザの互換性**
   - 古いブラウザでは`<details>`タグがサポートされていない場合があります
   - 最新のChrome、Firefox、Safari、Edgeでは問題なく動作します

### ボタンがクリックできない場合

1. **リンクのURLが正しく設定されているか確認**
   - ステップ7で説明したリンク設定を確認

2. **電話番号リンクの形式を確認**
   - `tel:05054823433`（ハイフンなしで統一）

## 次のステップ

ページが公開できたら、以下を実施しましょう：

### 1. **メニューに追加**
- 「外観」→「メニュー」
- 「相談の流れ」ページをメインメニューに追加
- 適切な順序に並び替え

### 2. **トップページからリンク**
- トップページに「相談の流れ」へのリンクを追加
- CTAボタンや案内文から誘導

### 3. **内部リンクの設置**
- 他のページ（事例詳細、費用・料金など）から「相談の流れ」へリンク
- サイト内の導線を整備

### 4. **スマホでの表示確認**
- 実際のスマートフォンで表示を確認
- ボタンのタップが正しく動作するか確認
- タイムラインが縦並びになっているか確認

### 5. **アクセス解析の設定**
- Google Analyticsで「相談の流れ」ページの閲覧を追跡
- CTAボタンのクリック率を計測（イベントトラッキング）

## よくある質問

### Q. CSSは毎回追加する必要がありますか？

A. いいえ、CSSは最初の1回だけ追加すればOKです。次回以降、他のページを作る場合も、この「相談の流れ」用のCSSは残しておいてください。

### Q. リンク先のLINE公式アカウントがまだない場合は？

A. 一旦 `href="#"` のままで公開して、LINE公式アカウント開設後にリンクを修正してください。または、LINEボタン自体をコメントアウト（削除）してもOKです。

### Q. 電話番号を変更したい場合は？

A. HTMLコード内の `tel:05054823433` と `050-5482-3433` を一括置換で変更してください。ハイフンあり/なし両方を変更する必要があります。

### Q. ステップの順序を変更できますか？

A. はい、HTMLコード内の`<section class="flow-step">`ブロックの順序を入れ替えることで変更できます。ただし、タイムラインの表示も合わせて修正する必要があります。

### Q. FAQの質問を追加/削除したい場合は？

A. HTMLコード内の`<details class="faq-item">`ブロックをコピーして追加、または削除してください。
