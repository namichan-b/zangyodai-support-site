# 残業代特化サイト - 参考資料

## デザイン原則スキルの使用方法

**すべてのコンテンツ作成・デザイン改善時は、必ず `design-principles` スキルを起動してください：**

```
/design-principles
```

スキルが起動すると、Claude Codeは以下の知識を自動参照します：
- 『ノンデザイナーズ・デザインブック』の4つの基本原則（近接・整列・反復・コントラスト）
- タイポグラフィの基本ルールと活字のカテゴリー
- 色彩理論とCMYK/RGBの使い分け
- 残業代特化サイトへの具体的な適用例

### デザインの4つの基本原則

1. **近接 (Proximity)**: 関連する情報をグループ化し、整理する
2. **整列 (Alignment)**: 要素を意図的に揃え、秩序と統一感を生み出す
3. **反復 (Repetition)**: デザイン要素を繰り返し、一貫性と視覚的な面白さを強化する
4. **コントラスト (Contrast)**: 要素間の違いを強調し、読者の視線を引きつけ、重要な部分を際立たせる

参照ファイル: `サイト作成知識/レポート：『ノンデザイナーズ・デザインブック』完全解説序章：デザインの目覚め —.md`

### マニュアルの読み込み方法

Claude Codeは **Pythonスクレイピング** でSWELL公式マニュアルを読み込めます：

response = requests.get(url)
soup = BeautifulSoup(response.content, 'html.parser')
article = soup.find('article')
print(article.get_text())

## 弁護士広告規制の遵守 (最重要)

**すべてのコンテンツ作成・編集時は、必ず弁護士広告規制を遵守してください。**

### 規制の根拠
- 弁護士法
- 弁護士職務基本規程
- 日本弁護士連合会「弁護士等の業務広告に関する規程」
- 日本弁護士連合会「業務広告に関する指針」
- 景品表示法（優良誤認表示・有利誤認表示の禁止）

詳細: `弁護士広告規制/広告規制まとめ.md` 参照

### 必須表示事項 (広告規程第9条)

すべてのページに以下を明記:
- 弁護士氏名: 川波晃生
- 所属弁護士会: 福岡県弁護士会

### 絶対に禁止される表現

#### 1. 事実に合致しない広告 (第1号)
- 虚偽の経歴・実績
- 架空の依頼者推薦文
- 実在しない「〇〇相談センター」等の名称

#### 2. 誤導・誤認のおそれのある広告 (第2号)
- 「専門分野」(客観的基準なし) → 「得意分野」は可
- 「日本一」「最も」等の最上級表現
- 「割安」「業界最低水準」等の曖昧な費用表現
- 同様の結果が得られると示唆する表現

#### 3. 誇大・過度な期待を抱かせる広告 (第3号)
- 「どんな事件でも解決」
- 「100%勝訴」「常勝」「不敗」
- 「必ず〇〇できます」等の結果保証
- 「たちどころに解決」

#### 4. 過度な不安をあおる広告 (第4号)
- 「今すぐ〇〇しないと権利が失われる」と過度に強調
- 依頼しなかった場合の悲惨な体験談

#### 5. 比較広告 (第5号)
- 他の弁護士・法律事務所との比較
- 「〇〇法律事務所よりも安く」等

#### 6. 法令・会則違反広告 (第6号)
- 非弁提携を示唆する表現

#### 7. 品位・信用を損なう広告 (第7号)
- 低俗・攻撃的な表現
- 「法の抜け穴」等の脱法行為を示唆
- 奇抜・派手・不快感を与える表現

### 表示禁止情報 (広告規程第4条)

以下は原則表示禁止:
1. 訴訟の勝訴率
2. 顧問先・依頼者名 (書面同意がある場合を除く)
3. 受任中の事件
4. 過去の事件 (依頼者が特定されず、利益を損なわない場合のみ可)

### 許容される表現

- 「得意分野」(主観的評価)
- 「〇〇に強い」(客観的根拠がある場合)
- 明確で詳細な料金体系
- 事務所の取組姿勢・プロセス・経験の説明
- 法的期限・結果を客観的かつ冷静に説明
- 完全匿名化された解決事例 (架空の場合はその旨明記)

### コンテンツチェック時の確認事項

1. 弁護士氏名・所属弁護士会が表示されているか
2. 誇大表現・最上級表現がないか
3. 結果を保証する表現がないか
4. 他事務所との比較がないか
5. 過度な不安を煽る表現がないか
6. 料金表示が明確か
7. 依頼者が特定される情報がないか
8. プロフェッショナルで品位ある表現か

### 違反のリスク

広告規程違反は:
- 弁護士会による中止命令
- 違反事実の公表 (信用失墜)
- 懲戒処分 (戒告・業務停止・退会命令・除名)

**新規コンテンツ作成時も、既存コンテンツ編集時も、必ず上記規制を確認してください。**
