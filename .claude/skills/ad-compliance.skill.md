# 弁護士広告規制コンプライアンスチェックスキル

## 概要
このスキルは、残業代特化サイトのコンテンツが弁護士広告規制に違反していないかをチェックします。

## 使用タイミング
- 新規コンテンツを作成した後
- 既存コンテンツを編集した後
- サイト全体のコンプライアンスチェックを実施したい時
- WordPressへのアップロード前の最終確認

## チェック項目

### 1. 必須表示事項 (広告規程第9条)
すべてのページに以下が明記されているか:
- [ ] 弁護士氏名: 川波晃生
- [ ] 所属弁護士会: 福岡県弁護士会

### 2. 禁止表現チェック (広告規程第3条)

#### 第1号: 事実に合致しない広告
- [ ] 虚偽の経歴・実績がないか
- [ ] 架空の依頼者推薦文がないか
- [ ] 実在しない「〇〇相談センター」等の名称を使用していないか

#### 第2号: 誤導・誤認のおそれのある広告
- [ ] 「専門分野」という表現を使用していないか (「得意分野」は可)
- [ ] 「日本一」「最も」「最高」等の最上級表現がないか
- [ ] 「割安」「格安」「業界最低水準」等の曖昧な費用表現がないか
- [ ] 「あなたも同じ結果が得られます」等、同様の結果を示唆する表現がないか

#### 第3号: 誇大・過度な期待を抱かせる広告
- [ ] 「どんな事件でも解決」等の表現がないか
- [ ] 「100%勝訴」「常勝」「不敗」等の表現がないか
- [ ] 「必ず〇〇できます」等の結果保証表現がないか
- [ ] 「たちどころに解決」等の過度な表現がないか

#### 第4号: 過度な不安をあおる広告
- [ ] 「今すぐ〇〇しないと権利が失われる」と過度に強調していないか
- [ ] 依頼しなかった場合の悲惨な体験談がないか

#### 第5号: 比較広告
- [ ] 他の弁護士・法律事務所との比較表現がないか
- [ ] 「〇〇法律事務所よりも安く/早く/確実に」等の表現がないか

#### 第6号: 法令・会則違反広告
- [ ] 非弁提携を示唆する表現がないか

#### 第7号: 品位・信用を損なう広告
- [ ] 低俗・攻撃的な表現がないか
- [ ] 「法の抜け穴」等の脱法行為を示唆する表現がないか
- [ ] 奇抜・派手・不快感を与える表現がないか
- [ ] プロフェッショナルで品位ある表現になっているか

### 3. 表示禁止情報チェック (広告規程第4条)

- [ ] 訴訟の勝訴率を表示していないか
- [ ] 顧問先・依頼者名を無許可で表示していないか
- [ ] 受任中の事件を表示していないか
- [ ] 過去の事件で依頼者が特定される情報がないか

### 4. 景品表示法チェック

- [ ] 優良誤認表示(サービス品質の誇大表現)がないか
- [ ] 有利誤認表示(費用・料金の誤認表現)がないか

## 実行手順

1. `WordPress用コンテンツ/` フォルダ内の全ファイルを読み込む
2. 各チェック項目に照らして違反表現がないか確認
3. 違反または疑わしい表現を発見した場合:
   - ファイル名
   - 該当箇所
   - 違反理由
   - 修正案
   を報告する
4. 問題がない場合は、その旨を報告する
