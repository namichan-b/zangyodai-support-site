# Webサイトページ作成スキル

## 概要
残業代特化サイトのコンテンツページを、HTML+CSSセットとして自動生成するスキル。
弁護士広告規制に完全準拠し、プロフェッショナルなデザインのページをコピペだけで作成できる。

## 対応ページタイプ

### 1. 事例詳細ページ
- 依頼者プロフィール
- 相談前の状況
- 解決までの流れ
- 回収金額と結果
- 学べるポイント
- FAQ
- CTA（相談案内）

### 2. 相談の流れページ

- 視覚的にわかりやすいデザイン

### 3. 費用・料金ページ

- 費用例の提示

### 4. 弁護士紹介ページ

## 出力するファイル

各ページタイプについて、以下3ファイルを生成：

1. **`{ページ名}_HTML.html`**
   - WordPress カスタムHTMLブロック用
   - セマンティックなHTML構造
   - アクセシビリティ対応

2. **`{ページ名}_CSS.css`**
   - WordPress 追加CSS用
   - レスポンシブ対応（PC/タブレット/スマホ）
   - プロフェッショナルなデザイン
   - 事例ページと統一感のあるカラースキーム

3. **`{ページ名}_WordPress貼り付け手順.md`**
   - ステップバイステップの手順書
   - トラブルシューティング
   - スクリーンショット付き説明

## デザインコンセプト

### デザイン原則の適用（重要）

**このスキル実行時は、必ず `design-principles.skill.md` のデザイン理論を参照すること**

参照ファイル：`.claude/skills/design-principles.skill.md`

#### 4つの基本原則の適用

すべてのページ作成時に、以下の4原則を必ず適用：

1. **近接 (Proximity)**: 関連情報をグループ化し、グループ間に十分な空白を確保
2. **整列 (Alignment)**: すべての要素を意図的に配置し、強い整列線（左揃え）を形成
3. **反復 (Repetition)**: フォント、色、ボタンスタイル、レイアウトパターンを全ページで統一
4. **コントラスト (Contrast)**: 見出しと本文のサイズ、ウェイト、色を劇的に変えて階層を明確化

詳細な理論と実践例は `design-principles.skill.md` を参照。

### カラースキーム（統一）
- **メインカラー**: ミドルブルー（#2874a6） - 信頼感・爽やかさ・プロフェッショナル
- **グラデーション**: ライトミディアム（#5499c7） - 明るさと親しみやすさ
- **アクセントカラー**: ゴールド（#c9a961） - 高級感・成功
- **背景色**: ホワイト・ライトグレー - 清潔感
- **テキスト**: ダークグレー（#333） - 可読性

### フォント
- **見出し**: 明朝体（ヒラギノ明朝 ProN） - 重厚感
- **本文**: ゴシック体（游ゴシック） - 読みやすさ
- **ボタン**: ゴシック体太字 - 視認性

### レイアウト
- 最大幅：1100px（読みやすさ重視）
- 余白を十分に確保
- セクション間の明確な区切り
- ボタンや重要情報の強調
- **行間**: コンパクトに（無駄な余白を削減）
- **スマホ表示**: 文字サイズを十分に確保（最小16px以上）
- **3カラムレイアウト**: 解決方法カード（交渉・労働審判・訴訟）は3カラムで横並びに表示

## サイト全体のマーケティング方針（重要）

### 集客ターゲット

**依頼見込み客の集客に注力**

- ❌ **相談のみの客は不要**: 収益につながらず、対応コストが負担となるため
- ✓ **依頼見込み客を優先**: 実際に受任につながる可能性の高い相談者を集客
- ✓ **将来的な相談料有料化の可能性**: 相談のみの客を減らし、質の高い相談に注力

### バッジ・メッセージの方針

- ✓ 「初回相談無料」: 入口のハードルを下げる
- ✓ 「着手金0円」: 費用面の不安を軽減
- ❌ 「相談のみでも大歓迎」: 使用しない（依頼見込みのない相談者を集めるため）

### コンテンツ作成時の注意

- 「気軽にご相談ください」ではなく、「まずはご相談ください」
- 相談から受任までの流れを明確に示す
- 解決事例や実績を強調し、依頼の価値を伝える
