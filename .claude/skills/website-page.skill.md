# Webサイトページ作成スキル

## 概要
残業代特化サイトのコンテンツページを、HTML+CSSセットとして自動生成するスキル。
弁護士広告規制に完全準拠し、プロフェッショナルなデザインのページをコピペだけで作成できる。

## 対応ページタイプ

### 1. 事例詳細ページ
- 依頼者プロフィール
- 相談前の状況
- 解決までの流れ
- 回収金額と結果
- 学べるポイント
- FAQ
- CTA（相談案内）

### 2. 相談の流れページ

- 視覚的にわかりやすいデザイン

### 3. 費用・料金ページ
- 料金表（見やすい表形式）
- 完全成功報酬制の説明
- 費用例の提示

### 4. 弁護士紹介ページ
- プロフィール
- 経歴

- メッセージ

### 5. よくある質問（FAQ）ページ
- Q&A形式

- その他

## 出力するファイル

各ページタイプについて、以下3ファイルを生成：

1. **`{ページ名}_HTML.html`**
   - WordPress カスタムHTMLブロック用
   - セマンティックなHTML構造
   - アクセシビリティ対応

2. **`{ページ名}_CSS.css`**
   - WordPress 追加CSS用
   - レスポンシブ対応（PC/タブレット/スマホ）
   - プロフェッショナルなデザイン
   - 事例ページと統一感のあるカラースキーム

3. **`{ページ名}_WordPress貼り付け手順.md`**
   - ステップバイステップの手順書
   - トラブルシューティング
   - スクリーンショット付き説明

## デザインコンセプト

### カラースキーム（統一）
- **メインカラー**: ネイビーブルー（#1a4d8f） - 信頼感・プロフェッショナル
- **アクセントカラー**: ゴールド（#c9a961） - 高級感・成功
- **背景色**: ホワイト・ライトグレー - 清潔感
- **テキスト**: ダークグレー（#333） - 可読性

### フォント
- **見出し**: 明朝体（ヒラギノ明朝 ProN） - 重厚感
- **本文**: ゴシック体（游ゴシック） - 読みやすさ
- **ボタン**: ゴシック体太字 - 視認性

### レイアウト
- 最大幅：900px（読みやすさ重視）
- 余白を十分に確保
- セクション間の明確な区切り
- ボタンや重要情報の強調

## 使用方法

### 基本的な呼び出し

```
/website-page
```

スキル起動後、以下を対話形式で確認：

1. **ページタイプの選択**
   - 事例詳細 / 相談の流れ / 費用・料金 / 弁護士紹介 / FAQ / その他

2. **コンテンツの指定**
   - 既存テキストファイルのパス
   - または、内容を直接入力

3. **カスタマイズ要望**
   - 特別なデザイン要素
   - 追加セクション

## CSS管理方針

### 初回のみ追加
- ページタイプごとのCSS
- 追加CSS欄に1回だけ追加

### 2回目以降
- HTMLのみ変更
- CSSは既存のものを利用
- 必要な場合のみCSSを更新

## スキル実行時の動作

1. ページタイプを確認
2. 既存コンテンツの有無を確認
3. 内容を読み込み/ヒアリング

5. HTML生成
6. CSS生成（初回のみ、または必要時のみ）
7. 手順書生成
8. ファイル保存
9. 完了報告と次のステップ案内
