# コンテンツ・SEOギャップ分析（福岡×残業代特化）

目的：主要クエリ（福岡 残業代 弁護士／ドライバー／小売店長 ほか）での上位獲得とCV増強のため、競合との差を可視化し、優先施策を決定する。

---

## 1. コンテンツギャップ（不足/強化余地）
- 事例の量と体系化：
  - ギャップ：交渉/審判/訴訟の類型別・職種別の事例集が不足。
  - 対応：職種×争点×手続のマトリクスで事例テンプレを量産。
- 職種特化の深掘り：
  - ギャップ：ドライバーの荷待ち・待機時間、店長の管理監督者性チェック等の深度が不足。
  - 対応：証拠の集め方（タコ/ドラレコ/GPS/ゲート入退館、任命辞令/鍵開閉時刻）を実務的に提示。
- FAQの充実：
  - ギャップ：検索意図を網羅するQ&Aが不足。
  - 対応：高頻度質問（匿名可否・在職中の請求・証拠が弱い場合）を短文で網羅、FAQ構造化データ。
- 地域特化情報：
  - ギャップ：福岡の労働事情・統計・機関案内のローカル性が薄い。
  - 対応：福岡の統計・労基署・労働局案内、地名×事例で内的関連性強化。

## 2. SEOギャップ（技術/内部/外部）
- 内部リンク：
  - ギャップ：争点記事→職種LP→相談の流れ/費用の三角導線が弱い。
  - 対応：関連記事ブロックと本文内CTAで相互リンク設計。
- 構造化データ：
  - ギャップ：FAQ/Article/LocalBusiness/LegalServiceの実装不足。
  - 対応：主要ページにスキーマ実装、Breadcrumbも整備。
- ページ速度（モバイル）：
  - ギャップ：LCP/CLSの課題（画像最適化/遅延読込）が想定。
  - 対応：画像WebP/遅延読込/critical CSS/フォント最適化。
- 外部シグナル：
  - ギャップ：地域メディア/専門サイトからの言及・被リンクが不足。
  - 対応：寄稿・監修・プレス/イベント露出でナチュラルな被リンク獲得。

## 3. CV導線ギャップ
- CTA配置：
  - ギャップ：FV固定CTA・スクロール追従が未整備。
  - 対応：TEL/LINE/短フォームの3系統を常時表示。
- フォーム摩擦：
  - ギャップ：項目過多、入力補助不足。
  - 対応：必須最少化（10項目以内）、選択肢化、途中保存/自動補完。
- 料金表示：
  - ギャップ：税込主表示、最低成功報酬、段階移行の精算明記の徹底不足。
  - 対応：料金ページ再設計＋LPにも要点サマリを表示。

## 4. 優先施策（Impact×Effort）
- 高Impact / 低Effort（先行）
  - LPにFV固定CTA（TEL/LINE/短フォーム）を実装
  - 料金ページの税込主表示・最低成功報酬・差額精算の明記
  - 争点→LP→相談の流れ/費用の内部リンク網
  - FAQの作成＋構造化データ実装
- 高Impact / 中Effort
  - 職種×争点×手続 事例テンプレの量産（まず6本：ドライバー3／店長3）
  - 証拠の集め方ページ（写真例・チェックリストDL）
  - Article/LegalService/LocalBusiness/Breadcrumbのスキーマ実装
- 中Impact / 低Effort
  - 既存記事の見出し最適化（1見出し1検索意図）と内部リンク追加
  - 画像の軽量化（WebP）と遅延読込
- 中Impact / 中Effort
  - 地域特化コンテンツ（福岡の統計・機関案内）
  - 外部露出（監修・寄稿・イベント）で言及獲得

## 5. 次アクション（2週間）
- W1：LPのCTA実装／料金ページ改訂／FAQ10問＋構造化
- W2：事例テンプレ6本草稿／証拠の集め方（ドライバー/店長）／内部リンク整備

指標：LP CVR、問い合わせ→面談率、主要クエリ順位、FAQクリック率、事例ページ到達率。
