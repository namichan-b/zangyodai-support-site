# SWELL実装ガイド - 残業代サポートサイト

## 🔧 初期設定

### 1. SWELLテーマの有効化
1. WordPress管理画面 → 外観 → テーマ
2. SWELLを有効化
3. 子テーマ（SWELL CHILD）も有効化推奨

### 2. SWELL基本設定
**カスタマイザー（外観 → カスタマイズ）で設定**

#### サイト全体設定
- **基本カラー**:
  - メインカラー: `#2874a6`（ミドルブルー）
  - テキストカラー: `#333333`
  - リンクカラー: `#2874a6`
  - リンクホバー: `#c9a961`（ゴールド）

#### 基本デザイン
- **フォント設定**:
  - 游ゴシック > ヒラギノ角ゴ > メイリオ
  - 本文サイズ: 16px（PC）/ 15px（SP）

## 📄 固定ページの作成方法

### 基本的な流れ
1. 固定ページ → 新規追加
2. タイトル入力
3. ブロックエディタで内容作成
4. ページ属性設定
5. 公開

### SWELLブロックの使い方

#### 1. フルワイドブロック（ヒーローエリア）
```
用途: トップページのメインビジュアル
設定:
- 背景: グラデーション or 画像
- 上下パディング: 大
- コンテンツ幅: 記事幅
```

#### 2. カラムブロック（横並びレイアウト）
```
用途: サービス紹介、料金表など
設定:
- PC: 2〜3カラム
- SP: 1カラム（自動）
- 間隔: 20px
```

#### 3. SWELLボタン
```
用途: CTA、お問い合わせリンク
設定:
- スタイル: 立体
- サイズ: 大
- カラー: カスタム（#c9a961）
- アイコン: 任意
```

#### 4. ステップブロック
```
用途: 相談の流れページ
設定:
- スタイル: ビッグ
- 形状: 四角
- ステップ番号表示: ON
```

#### 5. FAQブロック
```
用途: よくある質問
設定:
- スタイル: シンプル
- アイコン: Q&A
- 初期状態: 閉じる
```

#### 6. アコーディオンブロック
```
用途: 詳細情報の折りたたみ
設定:
- スタイル: シンプル
- アイコン: プラス/マイナス
```

## 🎨 カスタムCSS適用方法

### 方法1: カスタマイザーから（推奨）
1. 外観 → カスタマイズ → 追加CSS
2. CSSコードを貼り付け
3. 公開

### 方法2: 各ページごと
1. 固定ページ編集画面
2. SWELL設定 → CSS用コード
3. ページ固有のCSSを記入

### 方法3: 子テーマのstyle.css
1. wp-content/themes/swell_child/style.css
2. 全ページ共通のカスタマイズ

## 📱 レスポンシブ設定

### SWELLの自動対応
- **カラム**: SP時は自動で縦並び
- **文字サイズ**: デバイスごとに最適化
- **余白**: 自動調整

### 手動調整が必要な場合
```css
/* PC表示 */
@media (min-width: 960px) {
  .custom-class {
    /* PC用スタイル */
  }
}

/* スマホ表示 */
@media (max-width: 959px) {
  .custom-class {
    /* SP用スタイル */
  }
}
```
