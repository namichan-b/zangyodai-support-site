#### ❌ 絶対に禁止される表現

**1. 事実に合致しない広告（第1号）**
- [ ] 虚偽の経歴・実績がないか
- [ ] 架空の依頼者推薦文がないか

**2. 誤導・誤認のおそれのある広告（第2号）**
- [ ] 「専門分野」の表記がないか（「得意分野」はOK）
- [ ] 「日本一」「最も」等の最上級表現がないか
- [ ] 「割安」「業界最低水準」等の曖昧な費用表現がないか
- [ ] 同様の結果が得られると示唆する表現がないか

**3. 誇大・過度な期待を抱かせる広告（第3号）**
- [ ] 「どんな事件でも解決」がないか
- [ ] 「100%勝訴」「常勝」「不敗」がないか
- [ ] 「必ず〇〇できます」等の結果保証がないか
- [ ] 「たちどころに解決」がないか

**4. 過度な不安をあおる広告（第4号）**
- [ ] 「今すぐ〇〇しないと権利が失われる」と過度に強調していないか
- [ ] 依頼しなかった場合の悲惨な体験談がないか

**5. 比較広告（第5号）**
- [ ] 他の弁護士・法律事務所との比較がないか

**6. 品位・信用を損なう広告（第7号）**
- [ ] 「法の抜け穴」等の脱法行為を示唆する表現がないか
- [ ] 奇抜・派手・不快感を与える表現がないか

#### 📊 表示禁止情報（広告規程第4条）
- [ ] 訴訟の勝訴率の表示がないか
- [ ] 「解決率○○%」の表示がないか
- [ ] 顧問先・依頼者名の表示がないか（書面同意なし）
- [ ] 受任中の事件の表示がないか

## ✅ タスク10: LINE公式アカウント連携の設定

### 手順

#### 1. LINE公式アカウント作成（未作成の場合）
1. https://www.linebiz.com/jp/ にアクセス
2. 「LINE公式アカウントを作る」をクリック
3. アカウント情報を入力
   - アカウント名: 残業代サポート（Hi法律事務所）
   - 業種: 士業・専門サービス
4. アカウント作成完了

#### 2. 友だち追加URLを取得
1. LINE Official Account Manager にログイン
2. 「友だち追加」→「友だち追加URL」をコピー
   - 例: `https://lin.ee/xxxxxxx`

#### 3. QRコードを取得
1. 「友だち追加」→「QRコード」をダウンロード
2. 画像を保存

#### 4. WordPressにLINEボタンを追加

**追加箇所:**
- トップページ
- 相談の流れページ
- お問い合わせページ

**ボタンの設置方法:**
1. SWELLボタンブロックを追加
2. 設定:
   - リンク先: LINE友だち追加URL
   - ボタンテキスト: 「LINEで無料相談」
   - ボタンカラー: グリーン（#06C755）
   - アイコン: LINE
3. プレビューで確認
