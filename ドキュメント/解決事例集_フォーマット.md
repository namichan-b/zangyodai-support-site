作成日: 2025年10月22日

## 全体構成

### 2段階構成（一覧 + 個別詳細ページ）

```
【解決事例一覧ページ】（親ページ）
├─ 事例1の詳細ページ（子ページ）
├─ 事例2の詳細ページ（子ページ）
├─ 事例3の詳細ページ（子ページ）
└─ ...
```

**採用理由:**
- ✅ SEO効果が高い（「業種 + 残業代 + 事例」で個別ページが検索される）
- ✅ 読みやすい（興味のある事例だけ読める）
- ✅ 拡張性がある（事例が増えても対応しやすい）

## 依頼者プロフィール

- **年代**: 30代男性
- **勤続年数**: 3年
- **雇用形態**: 正社員
- **給与**: 月給30万円（固定残業代含むと説明されていた）

---

## 相談前の状況

Aさん（仮名）は大手小売チェーンで店長として勤務していましたが、月80時間を超える残業が常態化していました。

会社からは「給料に固定残業代が含まれている」と説明されており、どれだけ働いても給料は変わりませんでした。

退職を決意したとき、「このまま泣き寝入りしたくない」という思いで当事務所にご相談いただきました。

**相談前の悩み:**
- 月80時間以上の残業が常態化
- 固定残業代制だから諦めていた
- 退職後に請求できるか不安だった

---

## 解決までの流れ

### 1. 初回相談（無料）

就業規則と雇用契約書を確認したところ、固定残業代の要件を満たしていないことが判明しました。

**判明した問題点:**
- 固定残業代の時間数が明示されていない
- 基本給と固定残業代の区別が不明確
- 超過分の支払いに関する規定がない

### 2. 証拠収集

以下の証拠を収集しました：
- タイムカード（勤怠記録）
- シフト表
- 業務日報のメール
- 給与明細（3年分）

### 3. 会社との交渉

内容証明郵便で残業代の請求を開始。

当初、会社側は「固定残業代で支払い済み」と主張しましたが、証拠が明確だったため、交渉段階で和解が成立しました。

**交渉のポイント:**
- 客観的な証拠（タイムカード）が決め手
- 固定残業代の要件不備を明確に指摘
- 労働審判を視野に入れた交渉戦略

### 4. 解決

交渉開始から3ヶ月で和解が成立しました。

**残業代回収額: 120万円**

- 成功報酬: 約32万円（税込）
- **実際の手取り額: 約88万円**

### 解決方法
- **交渉による和解**（労働審判・訴訟には至らず）

### 解決期間
- **3ヶ月**

## この事例のポイント

✅ **固定残業代制でも諦めない**
固定残業代制度は、適切な要件を満たさなければ無効になります。

✅ **証拠の重要性**
タイムカードや業務メールなどの客観的証拠が交渉を有利に進めます。

✅ **交渉段階での早期解決**
労働審判や訴訟に至る前の解決で、成功報酬を抑えることができました。

✅ **退職後でも請求可能**
退職から3年以内であれば、残業代請求が可能です。

## 4. SEO対策のポイント

### ページタイトル（title タグ）
```
【解決事例】{業種}が{解決方法}で{金額}を回収 | 残業代サポート
```

**例:**
```
【解決事例】小売店長が交渉で120万円を回収 | 残業代サポート
```

### メタディスクリプション
```
{業種}で{勤続年数}勤務した{年代}が、固定残業代制でも{金額}の残業代を回収した事例。{解決方法}で{期間}で解決。福岡発・全国対応の残業代請求専門弁護士。
```

**例:**
```
小売店長として3年勤務した30代男性が、固定残業代制でも120万円の残業代を回収した事例。交渉で3ヶ月で解決。福岡発・全国対応の残業代請求専門弁護士。
```

### キーワード戦略
- 業種名 + 残業代 + 事例
- 業種名 + 残業代 + 回収
- 固定残業代 + 無効 + 事例
- 退職後 + 残業代 + 請求

## 5. WordPress掲載時の設定

### ページ階層
```
解決事例（親ページ）
├─ 【事例1】小売店長が...（子ページ）
├─ 【事例2】ドライバーが...（子ページ）
└─ 【事例3】飲食店スタッフが...（子ページ）
```

### カテゴリー・タグ設定
- **カテゴリー**: 解決事例
- **タグ**: 業種名、解決方法、回収金額レンジなど

### 内部リンク
- 事例一覧ページ → 各事例詳細ページへリンク
- 各事例詳細ページ → 「お問い合わせ」「費用・料金」へリンク
- 各事例詳細ページ → 事例一覧に戻るリンク

## 6. 守秘義務への配慮

### 匿名化の方法
- ✅ イニシャルまたは仮名（Aさん、Bさん）を使用
- ✅ 具体的な会社名は伏せる
- ✅ 年代は「30代」「40代」程度にとどめる
- ✅ 地域は「福岡県内」など広めに
- ✅ 金額は端数を丸める（120万円など）

### 掲載の同意
- ✅ 依頼者から事例掲載の同意を得る
- ✅ 「依頼者の声」は本人確認の上で掲載
