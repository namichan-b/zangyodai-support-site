# メインビジュアル リッチ化 - 設定手順書

残業代特化サイトのメインビジュアルをリッチでプロフェッショナルなデザインに変更する手順です。

---

## 📋 実装内容

### デザイン改善のポイント

1. **文字配置**: 左揃えに変更
2. **文字サイズ**: より大きく、インパクトのあるサイズに調整
3. **レイアウト**: よりリッチでプロフェッショナルなデザインに

### 具体的な変更内容

- メインタイトルを最大5remの大サイズに（レスポンシブ対応）
- テキストを左揃えに配置して洗練された印象に
- グラデーションオーバーレイで高級感を演出
- テキストシャドウで視認性と立体感を向上
- フェードインアニメーションで動的な演出

#### 確認・調整する項目

1. **メインビジュアルの高さ設定**
   - 推奨: 「数値で指定する」

2. **フィルター処理（オプション）**

## 🎨 オプション設定（お好みで調整）

CSSファイルの最後にコメントアウトされたオプション設定があります。

### オプション1: テキストエリアに半透明の背景を追加

```css
/* コメント /* */ を外して有効化 */
.p-mainVisual__textLayer {
    background: rgba(255, 255, 255, 0.1) !important;
    backdrop-filter: blur(10px) !important;
    padding: 2rem !important;
    border-radius: 12px !important;
}
```

**効果**: テキストの背景に磨りガラス風の半透明ボックスが表示され、より読みやすくなります。

### オプション2: テキストにアウトラインを追加

```css
/* コメント /* */ を外して有効化 */
.p-mainVisual__slideTitle.p-mainVisual__slideTitle {
    -webkit-text-stroke: 1px rgba(0,0,0,0.2);
}
```

**効果**: テキストに細いアウトラインが追加され、明るい背景でも視認性が向上します。

## 🔧 トラブルシューティング

### 問題1: CSSが反映されない

**解決方法**:
1. ブラウザのキャッシュをクリア（Cmd+Shift+R / Ctrl+Shift+F5）
2. WordPress.comのキャッシュをクリア:
   ```
   WordPress管理画面 > ツール > サイトのキャッシュをクリア
   ```
3. カスタマイザーで再度「公開」ボタンをクリック

### 問題2: テキストが見切れる・配置がおかしい

**解決方法**:
CSS内の `padding-left` と `padding-right` の値を調整：

```css
.p-mainVisual__textLayer {
    padding-left: 5% !important;  /* この値を変更 */
    padding-right: 5% !important; /* この値を変更 */
}
```

### 問題3: アニメーションが動作しない

**解決方法**:
アニメーションが邪魔な場合は、CSS内の以下のセクションをコメントアウト：

```css
/* 6. アニメーション効果（オプション） */
/* この部分全体をコメントアウト */
```

### 問題4: グラデーションが濃すぎる/薄すぎる

**解決方法**:
CSS内のグラデーション設定を調整：

```css
.p-mainVisual::before {
    background: linear-gradient(
        to right,
        rgba(0, 0, 0, 0.5) 0%,    /* この値を変更: 0.3-0.7 */
        rgba(0, 0, 0, 0.3) 50%,   /* この値を変更: 0.2-0.5 */
        rgba(0, 0, 0, 0.1) 100%   /* この値を変更: 0.0-0.3 */
    );
}
```
