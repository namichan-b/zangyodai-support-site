## 📋 設定手順

### ステップ1: SWELLカスタマイザーでドット削除

まず、SWELLのデフォルト設定でドットフィルターを無効化します。

```
WordPress管理画面
  ↓
外観 > カスタマイズ
  ↓
トップページ > メインビジュアル
  ↓
表示設定
```

#### フィルター処理の設定

「フィルター処理」という項目を探して：

- **フィルター処理**: 「なし」を選択
- もし「ドット」が選択されていたら「なし」に変更

#### オーバーレイカラーの設定

「オーバーレイカラー」の設定：

- **オーバーレイカラー**: 設定を削除（または透明に）
- **不透明度**: 0% に設定

これで、SWELLのデフォルトフィルターが無効化されます。

---

### ステップ2: 古いCSSを削除

現在「追加CSS」に入力されている古いCSSを **すべて削除** してください。

```
外観 > カスタマイズ > 追加CSS
  ↓
既存のCSSをすべて選択（Cmd+A / Ctrl+A）
  ↓
削除（Backspace / Delete）
```

---

### ステップ3: 新しいCSSを追加

プロジェクトフォルダの **`swell-professional-lawyer.css`** を開いてください。

```
ファイルの場所:
swell-professional-lawyer.css
```

#### CSSをコピー&ペースト

1. **全選択**: Cmd+A（Mac）/ Ctrl+A（Windows）
2. **コピー**: Cmd+C（Mac）/ Ctrl+C（Windows）
3. **カスタマイザーの「追加CSS」に貼り付け**: Cmd+V（Mac）/ Ctrl+V（Windows）

---

### ステップ4: プレビューで確認

右側のプレビュー画面で以下を確認：

✅ **ドットが消えている** - 背景がクリアに表示
✅ **グレーではなく色鮮やか** - 紺色の背景が美しく表示
✅ **タイトルが明朝体** - 格調高いフォントに
✅ **ゴールドの装飾ライン** - タイトル上部に細いライン
✅ **テキストが左揃え** - プロフェッショナルなレイアウト
✅ **ボタンがゴールド** - 高級感のあるCTAボタン

---

カスタマイザー上部の「公開」ボタンをクリック

## 🔧 トラブルシューティング

### 問題1: まだドットが表示される

**解決方法**:

1. カスタマイザーの「メインビジュアル > 表示設定」を再確認
2. 「フィルター処理」が「なし」になっているか確認
3. ブラウザのキャッシュをクリア（Cmd+Shift+R / Ctrl+Shift+F5）
4. WordPress.comのキャッシュクリア

### 問題2: まだグレーっぽい

**解決方法**:

1. 「オーバーレイカラー」の設定を確認
2. 不透明度が0%になっているか確認
3. 「グレースケール」フィルターが選択されていないか確認

### 問題3: CSSが反映されない

**解決方法**:

1. 古いCSSが完全に削除されているか確認
2. 新しいCSSが正しく貼り付けられているか確認
3. カスタマイザーで「公開」ボタンを押したか確認
4. ブラウザをリロード

### 問題4: フォントが明朝体にならない

**解決方法**:

使用しているデバイスに明朝体フォントがインストールされていない可能性があります。CSSには複数のフォールバックが設定されているため、環境に応じて最適なフォントが自動選択されます。

## 💡 カスタマイズオプション

### オプション1: 装飾ラインを非表示にする

タイトル上部のゴールドラインが不要な場合、CSS内の以下の部分を削除：

```css
/* 装飾ライン（オプション） */
body .p-mainVisual__slideTitle::before,
html body .p-mainVisual__slideTitle::before {
    /* この部分を削除またはコメントアウト */
}
```

### オプション2: ボタンの色を変更

ゴールド以外の色にしたい場合：

```css
body .p-mainVisual__slideBtn a {
    /* 例: ブルー系に変更 */
    background: linear-gradient(135deg, #1e3a8a 0%, #3b82f6 100%) !important;
    color: #ffffff !important;
}
```

### オプション3: グラデーションの濃さを調整

背景グラデーションの濃さを変更：

```css
background: linear-gradient(
    135deg,
    rgba(10, 35, 66, 0.75) 0%,    /* 0.85 → 0.75 で薄くなる */
    rgba(20, 50, 90, 0.6) 40%,    /* 0.7 → 0.6 */
    rgba(30, 60, 100, 0.4) 70%,   /* 0.5 → 0.4 */
    rgba(40, 70, 110, 0.2) 100%   /* 0.3 → 0.2 */
) !important;
```
